# Snakes-and-Ladders-board-game
A terminal-based implementation of the classic Snakes and Ladders board game, written entirely in C. The project features an automated simulation mode that runs multiple self-played games to track and calculate play statistics.
https://en.wikipedia.org/wiki/Snakes_and_ladders


## Features
* Print board (2D array)
* Place snakes and ladders (fixed and random locations)
* Dice rolling function
* Play function (accurate board movement and boundary wrapping)
* Winning condition evaluation
* Automated play simulation and testing cases

## Technical Concepts Applied
This project was built to demonstrate proficiency in:
* C Functions
* 2D Arrays & Grid Navigation
* Pointers Pass by Reference vs Pass by Value
* Random Number Generation & Time/Sleep delays
* Modulo Operator logic
